# Proxies

Proxy é o intermediário entre seu dispositivo e a internet. Cada tipo tem caso de uso:

- **Residencial**: IP de usuário real (mais difícil de detectar, mais caro)
- **Datacenter**: IP de servidor (rápido, barato, fácil de banir)
- **Mobile**: IP de rede móvel 3G/4G (caro, mas plataforma trata com mais permissividade)
- **ISP**: IP residencial estático (híbrido, cada vez mais popular)

A maioria dos casos de uso em marketing de performance combina antidetect browser + proxy (geralmente mobile ou residencial) por perfil.

## Quando usar

- Você opera múltiplas contas em plataformas que baniu IP de datacenter
- Precisa de IP de geolocalização específica pra testar anúncios locais
- Faz scraping ou monitoramento de sites
- Quer rodar anúncios em redes que exigem IP residencial (ex: Facebook com BM restrita)
