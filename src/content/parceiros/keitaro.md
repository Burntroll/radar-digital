# Keitaro

O **Keitaro** é provavelmente o tracker mais usado por afiliados brasileiros. Tem versão self-hosted (clássica) e versão cloud (Keitaro Cloud, mais nova). A interface é amigável, a documentação é robusta em português, e o suporte responde bem — combinação rara no nicho.

Tem S2S, integração com redes de afiliação, suporte a múltiplos domínios de tracking, filtros de bot, e ferramentas de split-test. Pra quem tá começando, é a curva de aprendizado mais suave entre os trackers profissionais.

## Pontos fortes

- Documentação em português
- Interface intuitiva (relativamente)
- Versão cloud elimina a dor de manter servidor
- Suporte responsivo
- Grande comunidade BR (fórum, grupos)

## Pontos de atenção

- Versão self-hosted tem custo único alto (licença perpétua)
- Versão cloud cobra por clicks, pode ficar caro em volume
- Alguns recursos avançados (custom reports) são limitados
